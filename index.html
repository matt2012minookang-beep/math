방정식 계산 사이트

1차, 2차, 고차방정식과 여러 미지수의 연립방정식을 계산할 수 있어.
한글 자판으로 잘못 입력한 영문도 자동 보정하고, 수학기호 버튼과 루트 버튼도 들어가 있어.
식 입력
입력 예시
• x^2=9
• x^3-6*x^2+11*x-6=0
• x+y=5
• 2*x-y=1
• √x=3
• √(x+1)=5
• 곱셈은 *, 나눗셈은 /, 거듭제곱은 ^
• 한글 자판으로 ㅌ을 입력하면 자동으로 x로 바뀜
자동 분석
식 개수
0
감지된 미지수
없음
판단
입력을 기다리는 중
입력 팁
√ 버튼을 누르면 루트가 들어가고,
/ 는 미리보기에서 분수처럼 보이고,
^ 는 지수로 보이게 렌더링돼.
결과

해

이 버전은 브라우저 한 파일 버전이라서 아주 복잡한 특수식은 한계가 있을 수 있지만, 일반적인 고차방정식과 기본적인 연립방정식은 계산 가능하게 구성했어.
