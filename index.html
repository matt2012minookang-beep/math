방정식 계산 사이트

1차, 2차, 고차방정식과 여러 미지수의 연립방정식을 계산할 수 있어.
왼쪽 식과 오른쪽 식을 입력하고 해 찾기를 누르면 돼. 한글은 자동 차단돼.
식 입력
입력 예시
• x^2=9
• x^3-6*x^2+11*x-6=0
• x+y=5
• 2*x-y=1
• (x+1)^2=16
• 루트는 x^1/2 또는 (x+3)^(1/2) 처럼 입력
• 곱셈은 *, 나눗셈은 /, 제곱/거듭제곱은 ^
자동 분석
식 개수
0
감지된 미지수
없음
판단
입력을 기다리는 중
결과

해

아직 계산하지 않았어.
이 버전은 브라우저 한 파일로 돌아가도록 만든 버전이라 매우 복잡한 비선형 연립방정식이나 특수한 경우는 한계가 있을 수 있어. 하지만 일반적인 고차방정식과 기본 연립방정식은 계산 가능하게 구성했어.
